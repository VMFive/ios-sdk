## 範例目的

透過這個範例的呈現, 我們可以利用 `NativeAd` 下載廣告素材, 並且只在列表中顯示廣告主題, 當使用者感興趣時,
我們再將影音主體彈跳出來, 既兼顧原先 UI / UX 設計, 也可以客制彈跳出的畫面.
